Алгоритмы. Числа Фибоначчи

Числа Фибоначчи — это наверное самая простая последовательность и практически каждый из нас ее знает: 1, 1, 2, 3, 5, 8, 13… Формально их можно отобразить просто:

F0 = 0, F1 = 1, Fn = Fn-2 + Fn-1, n >=2

Найти n-ое число Фибоначчи достаточно просто:

            
    function fibonacchi(n){
      if ( n == 0 ) return 0;

      if (n == 1) return 1;
      return fibonacchi(n-2) + fibonacchi(n-1);
    }
    console.log(fibonacchi(10));

Простая рекурсивная функция, которая вызывает сама себя, пока не дойдет до момента когда n = 0 или n = 1. Казалось бы, что тут сложного?
Числа Фибоначчи очень часто используют как объяснение такого подхода как мемоизация. Мемоизация — это подход, который позволяет нам сохранять результаты промежуточных решений, для того чтобы в следующих расчетах не повторять тоже самое.
Принцип невероятно прост, в данном случае мы выделяем массив — все значения которого == 0, и записываем все получившиеся вычисления, таким образом имеем следующий код: 

            
    var results = [0,1];

    function fibonacchi(n){
      if ( n == 0 ) return results[0];

      if (n == 1) return results[1];

      if (!results[n]){
        results[n] = fibonacchi(n-2) + fibonacchi(n-1);
      }
      return results[n];
    }
